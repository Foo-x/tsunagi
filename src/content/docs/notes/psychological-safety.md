- 心理的安全性 = 意見を率直に言い合える状態
- 話しやすいこと
    - 否定的な言葉ではなく、肯定的な言葉を使う
        - 「欠点」→「伸びしろ」
        - 「いい加減」→「おおらか」「寛大」
        - 「無口」「暗い」→「思慮深い」「穏やか」「落ち着いている」「控えめ」「冷静」
        - 「考えが浅い」→「楽観的」
        - 「頑固」→「一貫性がある」「信念がある」「意志が強い」
        - 「〇〇がダメです」→「△△するともっと良くなります」
        - 「いえ、～」→「なるほど、〇〇ということですね、ただ～」「〇〇だとすると～」
            - 否定を肯定と誤解されないように注意
    - 互いに敬意を払い、異なる意見を尊重する
    - すでに知っていることや自分と反対の意見でも話を遮らずに最後まで聞く
    - 話すことより聞くことを優先する
    - 体を相手に向けて目を見ながら聞く
    - コミュニケーション量を増やす
    - 不機嫌を相手にぶつけない
    - 良い言動は称賛する
        - 自分にとって当たり前であっても、その人は努力してやっている可能性がある
    - 5W1Hのうち、WHYは否定のニュアンスを感じさせることがあるので注意する
        - 「なぜそうしましたか」は「そうした理由がわからない」ということなので、正しくないことをしたように感じる
        - 「どのようにやりましたか」や「いつそれをやりましたか」などWHY以外を使うと和らぐ
    - 単刀直入に言うとキツくなるときは表現を和らげたりクッション言葉を挟んだりする
        - ネガティビティバイアスがあるため、ポジティブともネガティブともとれないメッセージはネガティブに感じやすい
        - 質問にする
            - 「ここが間違っています」→「ここが間違っていませんか」
            - 「〇〇してください」→「〇〇していただけますか」
        - Iメッセージにする
            - 「ここが間違っています」→「ここが間違っていると思います」「ここが間違っている気がします」
            - 「〇〇してください」→「〇〇していただけると助かります」「〇〇していただけますと幸いです」
        - 文末に「ね」「よ」「！」や絵文字を付ける
            - 「〇〇ですね」
            - 「〇〇ですよ」
            - 「〇〇です！」
            - 「〇〇です👍」
            - `...` は呆れているニュアンスを感じさせることがあるので注意する
            - 一部絵文字はネガティブなニュアンスを感じさせることがあるので注意する
        - 同じようなやり取りを複数回行う場合はテンプレートやフォーマットを共有しておく
            - 校正記号のイメージ
            - 効率も良くなる
            - 例
                - 誤字の修正は「誤字: 〇〇 → △△」
                - 修正しなくても良いが参考情報を提供するときは `[FYI]` を付ける
                - 修正してほしいときは `[MUST]` を付ける
                - 修正ではなくただの質問のときは `[QUESTION]` を付ける
        - 依頼するときのクッション言葉
            - 「お手数をおかけしますが」
            - 「お忙しいところ恐縮ですが」
            - 「恐れ入りますが」
            - 「差し支えなければ」
        - 厚意を断るときのクッション言葉
            - 「あいにくですが」
            - 「せっかくのご厚意ですが」
            - 「身に余るお話ではありますが」
            - 「心苦しいのですが」
        - 提案するときのクッション言葉
            - 「もしよろしければ」
            - 「ご迷惑でなければ」
- 助け合うこと
    - 「言わなくてもわかるだろう」とは考えずに対話する
    - 問題が発生したときに犯人探しをするのではなく、チームで解決する
    - 困ったときに助けを求める
    - 相手の要望に応えられないときは代替案を提示する
- 挑戦できること
    - アイディアを共有する
    - アイディアを否定するのではなくサポートする
    - 試行錯誤できる環境を作る
    - 建設的なフィードバックをすぐに行う
- 個性が受け入れられること
    - ハラスメントを行わない
        - 脅迫
        - ストーキング
        - 誹謗中傷、攻撃的な言動
        - 見下すような言動
        - 自らの優位性を主張する言動
        - 性的な言動
        - 不適切な物理的接触
        - ハラスメントにつながる撮影、録音
        - 許可なく個人情報を公開すること
        - 不当に他者を排除すること
    - チームの全員が活躍する機会を作る
    - 個人の思想を強制しない
        - 「社会人であれば休日も勉強すべき」など
